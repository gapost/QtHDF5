◆ metaTypeId()

int QH5Datatype::metaTypeId ( ) const

Get the Qt-metatype id corresponding to this datatype.

If getClass() returns QH5Datatype::STRING then this function returns QMetaType::QString.

Otherwise it calls H5Tget_native_type to obtain the native type of the HDF5 datatype and converts to an equivalent metatype id.
