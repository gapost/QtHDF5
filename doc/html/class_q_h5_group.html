QH5Group QH5Group::createGroup ( const char *  name,
bool  idxCreationOrder = false 
) const

Create a sub-group

A new group with a given name is created under the current group, unless an object with the same name exists or another error occurs.

If idxCreationOrder is set to true, then the creation order of items in the sub-group is registered and will be enumerable.

Parameters
nameName of the new group
idxCreationOrderOptional flag to register creation order
Returns
QH5Group The new group or an invalid object
