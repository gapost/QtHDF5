◆ open()

bool QH5File::open ( QIODevice::OpenMode  mode = QIODevice::ReadWrite)

Open the HDF5 file.

If the file does not exist it is created.

If the file exists but mode is QIODevice::Truncate then it is opened and truncated.

An existing HDF5 file can be opened either with QIODevice::ReadWrite or with QIODevice::ReadOnly.

Parameters
modeA combination of QIODevice flags
Returns
true If the file has been succesfully opened or created
false File does not exist or could not be created
